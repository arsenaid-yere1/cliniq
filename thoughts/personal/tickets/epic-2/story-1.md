# Epic 2 — Medical Document Processing

**Goal:** Convert MRI reports and chiropractor notes into structured clinical data.

## User Story 2.1 — MRI Report Extraction

As a provider
I want MRI findings automatically extracted
So that I can quickly review imaging results.

### Acceptance Criteria

- System extracts:
  - Body region
  - MRI date
  - Findings
  - Impression summary
  - Disc levels or anatomical findings
- Provider can view extracted findings
