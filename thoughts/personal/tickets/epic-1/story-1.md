# Epic 1 — Patient Case Management

**Goal:** Create and manage a patient case that aggregates MRI reports, chiropractor notes, visits, and billing.

## User Story 1.1 — Create Patient Case

As a clinic staff member
I want to create a patient case record
So that all clinical and billing information can be tracked for the patient.

### Acceptance Criteria

- User can create a patient with:
  - Name
  - DOB
  - Contact info
  - Case ID
  - Accident date (optional)
- System generates a unique Case Number
- Patient case dashboard is created
