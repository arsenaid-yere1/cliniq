# Epic 1 — User Story 1.3: Patient Document Repository

**As a** staff member
**I want to** upload medical documents to a patient case
**So that** they can be processed and used for documentation.

## Acceptance Criteria

- Accept PDF, DOCX, or image uploads
- User can label document type:
  - MRI report
  - Chiropractor report
  - Other
- Files stored securely
- File versioning supported
