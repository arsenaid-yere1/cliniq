# Epic 5 — Discharge Documentation

**Goal:** Create a discharge summary once treatment concludes.

## User Story 5.1 — Generate Discharge Summary

As a provider
I want a discharge note automatically created
So that the treatment episode is documented.

### Acceptance Criteria

- Note includes:
  - Summary of treatment
  - Number of PRP sessions
  - Outcomes
  - Functional improvement
  - Final diagnoses
  - Recommendations
