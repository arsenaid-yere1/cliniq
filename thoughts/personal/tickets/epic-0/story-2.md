# Epic 0 — Clinic Setup

**Goal:** Allow the provider to configure clinic information used across all documents and invoices.

## User Story 0.2 — Configure Provider Information

As a provider
I want to enter my professional details
So that clinical documents are attributed to me.

### Acceptance Criteria

- Provider can enter:
  - Provider full name
  - Credentials (MD, DO, DC, NP, etc.)
  - License number (optional)
  - NPI number (optional)
- This information appears in:
  - Document signatures
  - Procedure notes
  - Invoices
