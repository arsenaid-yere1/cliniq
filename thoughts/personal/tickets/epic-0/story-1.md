# Epic 0 — Clinic Setup

**Goal:** Allow the provider to configure clinic information used across all documents and invoices.

## User Story 0.1 — Configure Clinic Information

As a provider
I want to enter my clinic information
So that generated documents and invoices contain the correct clinic details.

### Acceptance Criteria

- Provider can configure:
  - Clinic name
  - Clinic address
  - Phone number
  - Email
  - Website (optional)
- This information appears automatically on:
  - Initial Visit Notes
  - PRP Procedure Notes
  - Discharge Notes
  - Medical Invoices
