# Epic 0 — Clinic Setup

**Goal:** Allow the provider to configure clinic information used across all documents and invoices.

## User Story 0.4 — Configure Provider Signature

As a provider
I want to upload my signature
So that finalized documents are signed automatically.

### Acceptance Criteria

- Provider can:
  - Upload signature image
  - Preview signature placement
- When a document is finalized, the following are displayed:
  - Provider name
  - Credentials
  - Signature
  - Timestamp
