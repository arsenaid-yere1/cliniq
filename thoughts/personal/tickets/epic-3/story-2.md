# Epic 3 — Initial Visit Documentation

**Goal:** Generate a clinical Initial Visit note using structured data.

## User Story 3.2 — Medical Necessity Generator

As a provider
I want medical necessity text automatically generated
So that documentation supports PRP treatment.

### Acceptance Criteria

- Narrative includes:
  - MRI findings
  - Conservative care history
  - Persistent symptoms
  - Functional impairment
- Provider can edit narrative before finalizing
