# Epic 3 — Initial Visit Documentation

**Goal:** Generate a clinical Initial Visit note using structured data.

## User Story 3.1 — Generate Initial Visit Draft

As a provider
I want an automatically generated initial visit note
So that I do not need to write documentation manually.

### Acceptance Criteria

- Document includes the following sections:
  - Patient info
  - Chief complaint
  - History of present illness
  - Imaging review
  - Prior treatment summary
  - Physical exam template
  - Assessment
  - Treatment plan
- Provider can edit text before finalizing
